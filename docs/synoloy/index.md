# 共享服务器

## 群晖 NAS 共享存储

公司使用群晖（Synology）NAS 设备提供集中式文件存储和共享服务。

### 主要功能

- **文件共享**：支持 SMB/CIFS、NFS、AFP 等多种协议
- **权限管理**：基于用户和群组的精细化访问控制
- **远程访问**：通过 QuickConnect 或 VPN 安全访问
- **数据备份**：自动备份重要数据，防止数据丢失
- **协同办公**：Synology Drive 实现团队文件协作

### 访问方式

#### Windows 访问
1. 打开文件资源管理器
2. 在地址栏输入 `\\NAS地址\共享文件夹`
3. 输入用户名和密码登录

#### macOS 访问
1. Finder → 前往 → 连接服务器
2. 输入 `smb://NAS地址/共享文件夹`
3. 输入凭据连接

### 常见问题

::: tip 无法连接
请检查网络连接是否正常，确认 NAS 设备已开机且网络可达。
:::

::: warning 权限不足
如提示权限不足，请联系管理员确认您的账户权限配置。
:::
