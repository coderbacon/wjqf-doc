# Kingdee System

## Kingdee Financial Management System

Kingdee is the financial management software used by the company to manage financial data and business processes.

### Main Modules

- **General Ledger**: Voucher entry, accounting processing, period-end closing
- **Reports**: Financial report generation and query
- **Fixed Assets**: Asset registration, depreciation calculation
- **Cash Management**: Cash and bank journal management
