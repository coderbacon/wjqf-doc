# About This Documentation

## Overview

The Wujiang Qingfeng Online Documentation Library is the company's internal knowledge management platform, providing employees with a unified and convenient documentation portal.

### Scope

- **Shared Server**: Synology NAS usage guide and FAQ
- **Kingdee System**: Financial system notices and operation manuals
- **DMS System**: Dealer Management System user guide

### How to Use

Browse documents by module through the top navigation bar and left sidebar. Use the search function to quickly find what you need.

### Updates & Maintenance

This documentation library is maintained by the IT department. Content is continuously updated. For questions or suggestions, please contact the administrator.
