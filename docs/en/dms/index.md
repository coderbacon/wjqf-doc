# DMS System

## Dealer Management System

DMS (Dealer Management System) is provided by the manufacturer and used primarily for business operations.

### Key Features

- **Customer Management**: Customer information entry, query, and maintenance
- **Vehicle Management**: Vehicle registration, maintenance and repair records
- **Order Management**: Creation and tracking of customer vehicle service orders
- **Inventory Management**: Parts inventory query and management
- **Reports**: Business data statistics and analysis

### Getting Started

#### Login
1. Open the DMS system login page
2. Enter the account and password provided by the manufacturer
3. Change your password after first login

#### Daily Operations
- Customer reception → Create work order → Assign task → Complete & settle
- Refer to each module's operation manual for more details

### FAQ

::: warning System Lag
If the system is slow to respond, check your network connection or avoid peak business hours.
:::

::: tip Data Sync
DMS data syncs with the manufacturer's server in real time. Ensure you wait for data submission to complete after each operation.
:::
