# DMS 系统

## 经销商管理系统

DMS（Dealer Management System）由厂家提供，主要用于公司业务层面管理。

### 主要功能

- **客户管理**：客户信息录入、查询和维护
- **车辆管理**：车辆信息登记、维修保养记录
- **订单管理**：客户车辆维修保养订单的创建与跟踪
- **库存管理**：配件库存查询和管理
- **报表统计**：业务数据统计与分析

### 快速开始

#### 系统登录

1. 打开 DMS 系统登录页面
2. 输入厂家提供的账号和密码
3. 首次登录后建议修改密码

#### 日常操作

- 接待客户 → 创建工单 → 派工 → 完工结算
- 更多详细操作请查阅各模块的操作手册

### 常见问题

::: warning 系统卡顿
如遇系统响应慢，请检查网络连接，或避开业务高峰期使用。
:::

::: tip 数据同步
DMS 数据与厂家服务器实时同步，请确保操作完成后等待数据提交成功。
:::
